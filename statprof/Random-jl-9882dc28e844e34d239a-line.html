124 """
125 Sampler(rng, x, repetition = Val(Inf))
126
127 Return a sampler object that can be used to generate random values from `rng` for `x`.
128
129 When `sp = Sampler(rng, x, repetition)`, `rand(rng, sp)` will be used to draw random values,
130 and should be defined accordingly.
131
132 `repetition` can be `Val(1)` or `Val(Inf)`, and should be used as a suggestion for deciding
133 the amount of precomputation, if applicable.
134
135 [`Random.SamplerType`](@ref) and [`Random.SamplerTrivial`](@ref) are default fallbacks for
136 *types* and *values*, respectively. [`Random.SamplerSimple`](@ref) can be used to store
137 pre-computed values without defining extra types for only this purpose.
138 """
139 Sampler(rng::AbstractRNG, x, r::Repetition=Val(Inf)) = Sampler(typeof_rng(rng), x, r)

302 """
303 rand([rng=GLOBAL_RNG], [S], [dims...])
304
305 Pick a random element or array of random elements from the set of values specified by `S`;
306 `S` can be
307
308 * an indexable collection (for example `1:9` or `('x', "y", :z)`),
309 * an `AbstractDict` or `AbstractSet` object,
310 * a string (considered as a collection of characters), or
311 * a type: the set of values to pick from is then equivalent to `typemin(S):typemax(S)` for
312 integers (this is not applicable to [`BigInt`](@ref)), to ``[0, 1)`` for floating
313 point numbers and to ``[0, 1)+i[0, 1)`` for complex floating point numbers;
314
315 `S` defaults to [`Float64`](@ref).
316 When only one argument is passed besides the optional `rng` and is a `Tuple`, it is interpreted
317 as a collection of values (`S`) and not as `dims`.
318
319
320 !!! compat "Julia 1.1"
321 Support for `S` as a tuple requires at least Julia 1.1.
322
323 # Examples
324 ```julia-repl
325 julia> rand(Int, 2)
326 2-element Array{Int64,1}:
327 1339893410598768192
328 1575814717733606317
329
330 julia> using Random
331
332 julia> rand(MersenneTwister(0), Dict(1=>2, 3=>4))
333 1=>2
334
335 julia> rand((2, 3))
336 3
337
338 julia> rand(Float64, (2, 3))
339 2×3 Array{Float64,2}:
340 0.999717 0.0143835 0.540787
341 0.696556 0.783855 0.938235
342 ```
343
344 !!! note
345 The complexity of `rand(rng, s::Union{AbstractDict,AbstractSet})`
346 is linear in the length of `s`, unless an optimized method with
347 constant complexity is available, which is the case for `Dict`,
348 `Set` and `BitSet`. For more than a few calls, use `rand(rng,
349 collect(s))` instead, or either `rand(rng, Dict(s))` or `rand(rng,
350 Set(s))` as appropriate.
351 """

378 """
379 seed!([rng=GLOBAL_RNG], seed) -> rng
380 seed!([rng=GLOBAL_RNG]) -> rng
381
382 Reseed the random number generator: `rng` will give a reproducible
383 sequence of numbers if and only if a `seed` is provided. Some RNGs
384 don't accept a seed, like `RandomDevice`.
385 After the call to `seed!`, `rng` is equivalent to a newly created
386 object initialized with the same seed.
387
388 If `rng` is not specified, it defaults to seeding the state of the
389 shared thread-local generator.
390
391 # Examples
392 ```julia-repl
393 julia> Random.seed!(1234);
394
395 julia> x1 = rand(2)
396 2-element Array{Float64,1}:
397 0.590845
398 0.766797
399
400 julia> Random.seed!(1234);
401
402 julia> x2 = rand(2)
403 2-element Array{Float64,1}:
404 0.590845
405 0.766797
406
407 julia> x1 == x2
408 true
409
410 julia> rng = MersenneTwister(1234); rand(rng, 2) == x1
411 true
412
413 julia> MersenneTwister(1) == Random.seed!(rng, 1)
414 true
415
416 julia> rand(Random.seed!(rng), Bool) # not reproducible
417 true
418
419 julia> rand(Random.seed!(rng), Bool)
420 false
421
422 julia> rand(MersenneTwister(), Bool) # not reproducible either
423 true
424 ```
